
# Lesson 2: Variables and Data Types

## Goal

Learn how to store information.

```typescript
let name = "Ganesh";
let age = 43;
let isStudent = true;
```

### Common Data Types

```typescript
let name: string = "Ganesh";
let age: number = 43;
let isStudent: boolean = true;
let value: any = "Anything";
let empty: null = null;
let notAssigned: undefined = undefined;
```

### Learn

- string
- number
- boolean
- any
- null
- undefined
